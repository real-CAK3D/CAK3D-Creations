# CAK3D-Creations project catalog

This document is the plain-language source list behind the public portfolio page. The public site should feel like a business portfolio, not a raw inventory dump, but these notes keep the project universe visible for future edits.

## Map-Pi

Trail and field-use app direction. Core themes: route planning, GPX/GeoJSON trail files, offline/PWA support, weather/last-known state, markers, field kit prep, and eventual Raspberry Pi deployment.

## Home Inventory

Phone-first family inventory app. Core themes: NFC routes, item/shelf/level/box hierarchy, Supabase-backed cloud state, Vercel deployment, photo handling, themes, user settings, and rollback backups.

## Ibby Auto Works

Auto-service business app direction. Core themes: customer portal, service request/intake flow, admin hub, payment planning, Supabase backend, Google/Stripe style integrations, and business operations tooling.

## MyPetID

Pet safety/profile app. Core themes: public pet profile, NFC finder scan flow, owner dashboard, care records, walks, medical/vet records, training/play logs, Dog Pack social concepts, and admin/user mode planning.

## Device dashboards

Simple dashboards for CAK3D devices and services: Garden VM, NukeBox, theBAK3RY, Hack-Safe, HP laptop, local previews, telemetry, ports, uptime, and status checks.

## Hermes / Garden command center

Agent-coordinated workshop operations. Core themes: Ganja as Discord front desk, The Gardiner as coordinator, specialist profiles for devices and projects, routed reports, safer approval paths, and low-clutter Discord lanes.

## Home Assistant / smart-home lane

Smart-home experiments through Homie and Home Assistant: room/device controls, household status, lights/switches/sensors, practical automations, and future dashboard views that tie house status into the broader workshop.

## Raspberry Pi projects

Raspberry Pi 5 systems, CrowPi/Hack-Safe, theBAK3RY, future Pi Zero 2 WH/P4wnagotchi work, field devices, kiosks, small local servers, and hardware-adjacent experiments.

## NukeBox / local app lab

The GMKTec NukeBox is a Windows/local-app workbench for heavier previews, Next.js/Vite apps, Tailscale testing, Discord workflows, DayZ edits, and experiments that later move to GitHub/Vercel.

## Knowledge, docs, and wiki systems

Project docs, GitHub wiki notes, Obsidian/knowledge-base ideas, reusable procedures, and handoff material so the growing workshop has memory instead of scattered one-off chats.

## Discord bots and automation

CAK3DBot roots, private Discord command lanes, specialist bot identities, channel organization, bot workflows, and dashboard-style reports.

## DayZ / CAD / creative tech roots

DayZ server work, Discord communities, CAD/digital design, projection mapping, music-production experiments, mechanic-style troubleshooting, and the DIY creative lane that started the whole thing.
