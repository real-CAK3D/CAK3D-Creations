# CAK3D-Creations project catalog

This document is the plain-language source list behind the public portfolio page.

## Map-Pi

Trail and field-use app direction. Core themes: route planning, GPX/GeoJSON trail files, offline/PWA support, weather/last-known state, markers, field kit prep, and eventual Raspberry Pi deployment.

## Home Inventory

Phone-first family inventory app. Core themes: NFC routes, item/shelf/level/box hierarchy, Supabase-backed cloud state, Vercel deployment, photo handling, themes, user settings, and rollback backups.

## Ibby Auto Works

Auto-service business app direction. Core themes: customer portal, service request/intake flow, admin hub, payment planning, Supabase backend, Google/Stripe style integrations, and business operations tooling.

## MyPetID

Pet safety/profile app. Core themes: public pet profile, NFC finder scan flow, owner dashboard, care records, walks, medical/vet records, training/play logs, Dog Pack social concepts, and admin/user mode planning.

## Device dashboards

Simple dashboards for CAK3D devices and services: Garden VM, NukeBox, theBAK3RY, Hack-Safe, HP laptop, local previews, telemetry, ports, uptime, and status checks.

## Raspberry Pi projects

Raspberry Pi 5 systems, CrowPi/Hack-Safe, theBAK3RY, future Pi Zero 2 WH/P4wnagotchi work, field devices, kiosks, small local servers, and hardware-adjacent experiments.

## Discord bots and automation

CAK3DBot roots, private Discord command lanes, specialist bot identities, channel organization, bot workflows, and dashboard-style reports.
